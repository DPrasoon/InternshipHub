<h1 align="center">Internship Hub</h1>
<h4 align="center">An online platform to help college students find internships</h4>

## About
A dynamic web application built using JavaEE stack and MySQL database

## Modules
* **Search**
* **Sign-Up**
* **Login**
* **Add Data**
* **Update Data**
* **Delete Data**
* **Contact & Feedback**
